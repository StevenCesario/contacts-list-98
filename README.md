The very first assigment for the Frontend course of my Full Stack medieinstitutet studies :)

It gave a very healthy and fun journey moving from my brain dumped fully working but all over the place spaghetti code with a God function to a more pragmatic orthogonal system. 

This in turn has become my blueprint for the second assignment with API integration (I used the Spotify and called it Vibe Salad, will be Public soon), my [Lock In Intentional Timer](https://github.com/StevenCesario/lock-in-intention-timer) and all future Full Stack project builds!

I like to believe that as long as there are clear pragmatic principles to a system, even if the scale is small like a contacts list app like this, the principles still stand when you scale it up to a larger system with more functions!

In that sense, this is one of the most important builds I will ever do and a rather perfect start to my Full Stack studies 🌱
